# 📂 Gestió de fitxers i directoris

| 🖥️ Comanda | 📌 Funció |
|---|---|
| `lc` | 📂 Mostra tots els fitxers i carpetes del directori actual |
| `cdir` | 📁 Canvia el directori actual a un altre especificat |
| `hcdir` | 🆕 Crea un nou directori amb el nom indicat |
| `eldir` | ❌ Elimina un directori especificat |
| `mc` | 📄 Mostra el contingut d’un fitxer en pantalla |

---

# ⚙️ Informació del sistema

| 🖥️ Comanda | 📌 Funció |
|---|---|
| `sos` | 🆘 Mostra l’ajuda i totes les comandes disponibles |
| `edicio` | 🏷️ Mostra la versió actual de NitroOS |
| `seemem` | 🧠 Mostra la memòria disponible i el consum actual |
| `tf` | ⏱️ Mostra el temps de funcionament del sistema |

---

# 🛠️ Utilitats

| 🖥️ Comanda | 📌 Funció |
|---|---|
| `lp` | 🧹 Neteja completament la pantalla |
| `scrib` | ✍️ Escriu text en pantalla o dins d’un fitxer |
| `adeu` | 🔴 Apaga el sistema |
| `fora` | 🔄 Reinicia el sistema |
